> Code du projet de l'UAA9 (examen) – Jeu de dominos en C#

---

## 📝 Description

Ce projet est un **jeu de dominos** développé en C# dans le cadre de l'UAA9.  
Il permet de jouer seul ou à plusieurs, gérer la pioche, poser ses dominos et suivre les scores.  

**Objectif :** Poser tous ses dominos avant les autres joueurs et gagner la partie !

---

## ✨ Fonctionnalités

- 🟢 Génération d’un **jeu complet de dominos**.  
- 🟢 **Distribution des dominos** aux joueurs.  
- 🟢 Vérification des **placements valides**.  
- 🟢 Gestion de la **pioche** si aucun coup possible.  
- 🟢 Affichage du **plateau de jeu** et des mains des joueurs.  
- 🟢 Calcul et affichage du **score final**.  
- 🟢 Mode plusieurs joueurs.
